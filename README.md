# analema
Analema: Pranata Mangsa, Anglaras Semesta adalah aplikasi reflektif berbasis alam yang menggabungkan informasi cuaca aktual dengan kearifan lokal Pranata Mangsa. Aplikasi ini membantu pengguna membaca tanda langit, memahami siklus musim, serta menyelaraskan aktivitas harian dengan ritme semesta.
